## 프로젝트 사용 기술

- **Language/Framework**
    - Java 17, Spring Boot 3.*
    - Spring JPA, Data MongoDB, QueryDSL
    - STOMP, SSE, Quartz, Elastic Search, Swagger
    - Rate Limiting, Web Scraping
    - JWT, OAuth 2.0
    - S3, FCM, Spring Mail
- **DB**
    - Redis
    - MongoDB
    - MySQL
- **Infra**
    - AWS -> On-Premise
    - Grafana/Prometheus
    - Docker, Github Actions
