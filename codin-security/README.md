# codin-security

> Spring Security 공통 설정 모듈

JWT 필터, 인증 핸들러, Security 설정 등 전 모듈이 공유하는 보안 컴포넌트를 제공합니다.

| 패키지 | 설명 |
|--------|------|
| `config` | SecurityFilterChain 설정 |
| `filter` | JWT 인증 필터 |
| `handler` | 인증/인가 실패 핸들러 |
| `jwt` | JWT 유틸리티 |
| `service` | UserDetailsService 구현 |
| `util` | 보안 관련 유틸리티 |
